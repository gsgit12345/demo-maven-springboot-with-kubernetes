@ComponentScan:-it is used to scan the package for a  bean.it is used with @Configuration annotation.
@Component: It is a class-level annotation. It is used to mark a Java class as a bean.
@EnableAutoConfiguration: It auto-configures the bean that is present in the classpath
@Configuration: It is a class-level annotation. The class annotated with @Configuration used by Spring Containers as a source of bean definitions.

@RestController:-if we annotate a class with this annotation.it means that this class will handle the http request.
@RequestMapping:-any http request having "/something" will land on this class.
